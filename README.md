# Weather App

A responsive weather application built with React and Tailwind CSS that shows current weather, hourly forecast, and 5-day forecast for any city.

## Features

* Search weather by city
* Current weather details
* Hourly forecast
* 5-day forecast
* Responsive UI
* Toast notifications for errors
* Real-time weather API integration

## Tech Stack

* React
* Tailwind CSS
* Vite
* OpenWeather API
* Sonner

## Installation

```bash
npm install
npm run dev
```

## Environment Variables

Create a `.env` file and add:

```env
VITE_WEATHER_API_KEY=your_api_key_here
```

## Dependencies

```bash
npm install axios react-icons sonner tailwindcss lucide-react
```
